Sass란?

Sass는 세상에서 가장 완성도 높고 안정적이며 강략한 CSS 확장 언어입니다. Sass 독자적인 구문(변수, 중첩 규칙, 함수, 컨디션, 루프, 믹스인 등)을 사용할 수 있는 환경을 제공합니다. 현재 CSS에서 변수를 사용할 수 있지만, 나머지 기능은 아직 사용할 수 없습니다.

  • CSS 호환성 Sass는 모든 CSS 버전과 완벽하게 호환됩니다.
  • 다양한 기능 Sass는 다른 어떤 CSS 확장보다 많은 기능과 능력을 자랑합니다.
  • 성장 / 관리 Sass는 약 16년간 코어 팀에 의해 적극적으로 유지되고 있습니다.
  • 인지도 Sass는 업계에서 최고의 CSS 확장 언어로 인정하고 있습니다.
  • 커뮤니티 Sass는 수백 명의 개발자로 구성된 여러 컨소시엄에서 적극적으로 지원하고 개발합니다.
  • 프레임워크 Sass를 기반으로 구축된 다양한 프레임워크가 존재합니다. (Compass, Bourbon, Susy 등)
